# Jubla-Reporting
Jubla Reporting Tool

[![Netlify Status](https://api.netlify.com/api/v1/badges/5f437261-5c16-4d69-87a1-6d7bb4cb87f6/deploy-status)](https://app.netlify.com/projects/jubla-wald-reporting/deploys)

## Setup

Dieses Projekt nutzt Netlify Functions und Google Sheets. Sensible Daten wie E-Mail-Adressen und Zugangsdaten werden über Umgebungsvariablen gesteuert.

### Environment

Lege eine Datei `.env` an oder konfiguriere die Variablen in Netlify. Eine Beispiel-Datei liegt in `.env.example`.

## Rechtliche Hinweise

Dieses Tool sammelt Daten zur Qualitätssicherung der Gruppenstunden der Jubla Wald ZH. Alle erfassten Daten werden vertraulich behandelt und ausschliesslich zur internen Nutzung innerhalb der Organisation verwendet.

- **Datenschutz:** Personenbezogene Daten werden nur in dem Umfang erhoben, wie es für die Funktion des Tools notwendig ist.
- **Zweck:** Das Tool dient der Erfassung von Gruppenstunden-Berichten und der automatischen Benachrichtigung bei kritischen Situationen.
- **Speicherung:** Daten werden in Google Sheets gespeichert und sind nur für autorisierte Personen zugänglich.
- **Löschung:** Berichte können auf Anfrage gelöscht werden.

Bei Fragen wenden Sie sich an die Scharleitung.
