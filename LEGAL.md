# Rechtliche Hinweise & Datenschutz

Dieses Dokument informiert über die rechtlichen Rahmenbedingungen und den Umgang mit Daten im Jubla-Reporting Tool.

## 1. Allgemeines
Dieses Tool dient der Qualitätssicherung der Gruppenstunden. Alle erfassten Daten werden vertraulich behandelt und ausschliesslich zur internen Nutzung innerhalb der Organisation verwendet.

## 2. Zweck der Datenverarbeitung
Das Tool dient der Erfassung von Gruppenstunden-Berichten und der automatischen Benachrichtigung der Scharleitung bei kritischen Situationen (Dynamik 4 oder 5). Dies ermöglicht eine zeitnahe Unterstützung der Leiterinnen und Leiter.

## 3. Datenspeicherung & Sicherheit
*   **Speicherort:** Die Berichte werden in Google Sheets gespeichert.
*   **Zugriff:** Nur autorisierte Personen haben direkten Zugriff auf die vollständige Tabelle.
*   **Übertragung:** Die Daten werden verschlüsselt (HTTPS) an das Backend übertragen.

## 4. Datenschutz (DSGVO / DSG)
Personenbezogene Daten werden nur in dem Umfang erhoben, wie es für die Funktion des Tools notwendig ist (z.B. Name des Melders).

### Google Sheets API
Zur Speicherung der Berichte wird die Google Sheets API genutzt. Dabei werden die Formulardaten an Google übermittelt.

## 5. Löschung & Auskunft
Berichte können auf Anfrage bei der Scharleitung korrigiert oder gelöscht werden.

## 6. Kontakt
Bei Fragen zum Datenschutz oder zur Nutzung des Tools wende dich bitte direkt an die Scharleitung.
